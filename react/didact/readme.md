## Concurrent Mode 并发模式
React Concurrent Mode 是一种让渲染过程可中断、可优先级排序的机制，
通过将工作拆分为小块并允许高优先级更新（如用户输入）插队，
从而避免主线程阻塞，提升应用的响应性和流畅度。
