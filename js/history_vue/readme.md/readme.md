#js 历史

-const 
 常量，es5 没有专门的常量声明
 简单数据类型 常量  不能重新赋值
 复杂数据类型 常量  可以重新赋值，但是不能修改属性值
 对象呢？值可以改变指向的内存地址不能发生改变。

-赋值
 简单数据类型 值传递  
 复杂数据类型  地址传递（引用式赋值 ）
 const 类型不同处理不同 

 内存栈 连续， 空间比较小 快
 内存堆 不连续 空间比较大 慢
 const 简单数据类型，复杂数据类型虽然表现不一致，内存栈中的值不会发生改变，但是内存堆中的值会发生改变。
 所以，const 定义的简单数据类型，复杂数据类型的内存堆中的值是可以改变的。
 简单数据类型  值传递  栈
 复杂数据类型  地址传递 堆
