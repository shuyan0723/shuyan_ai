## 概念定义
低代码或零代码是通过可视化的拖拽代替手写代码构建应用，低代码适合开发者提效，
零代码让非技术人员也能搭建简单应用，常用语表单、审批流程、数据看板等场景，快速满足业务需求
- 可视化 canvas
- 拖拽 好理解 工作流，程序流

## aisuda
Aisuda（爱速搭）是阿里巴巴推出的一款低代码应用搭建平台，旨在通过可视化拖拽等方式，
让开发者和业务人员能快速、高效地构建企业级应用。
